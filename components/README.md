# Components

This directory contains all React components organized by feature:

- `canvas/` - Canvas workspace and component rendering
- `editor/` - Editor layout and toolbar components
- `properties/` - Properties panel for component editing
- `library/` - Component library panel
- `prompt/` - Prompt editor and history
- `preview/` - Responsive preview components
- `export/` - Code export modal and utilities
- `suggestions/` - AI suggestions panel
- `tokens/` - Design token management UI
- `ui/` - Shared UI components (buttons, inputs, etc.)

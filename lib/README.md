# Library

This directory contains utilities, services, and core business logic:

- `ai/` - AI prompt engine and suggestion engine
- `services/` - External service integrations (OpenAI, Claude)
- `utils/` - Utility functions and helpers
- `db/` - Database utilities and Prisma client
- `auth/` - Authentication utilities
- `export/` - Code export engine
- `validation/` - Zod schemas and validation
